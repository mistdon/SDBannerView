详细描述

地图视图类（Overlay相关的操作）。

注解
Overlay 为覆盖物，提供：圆、折线、图片等类型的覆盖物。

Method Documentation

- (void) addOverlay: (id< MAOverlay >)  overlay

向地图窗口添加 Overlay。

注解
需要实现 MAMapViewDelegate::mapView:viewForOverlay: 函数来生成标注对应的 view。

默认添加层级：MAGroundOverlay默认层级为MAOverlayLevelAboveRoads，其余overlay类型默认层级为MAOverlayLevelAboveLabels

参数
overlay要添加的 Overlay。

Extends class MAMapView.
