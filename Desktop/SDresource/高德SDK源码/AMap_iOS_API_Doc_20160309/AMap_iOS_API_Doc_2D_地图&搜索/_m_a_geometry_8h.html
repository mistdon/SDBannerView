typedef void(^ AMapTileProjectionBlock)(int offsetX, int offsetY, int minX, int maxX, int minY, int maxY)

获取墨卡托投影切块回调block。

参数
offsetX左上点距离所属tile的位移X。
offsetY左上点距离所属tile的位移Y。
minX覆盖tile的最小x。
maxX覆盖tile的最大x。
minY覆盖tile的最小y。
maxY覆盖tile的最大y。

MACoordinateRegion MACoordinateRegionMakeWithDistance ( CLLocationCoordinate2D  centerCoordinate,
CLLocationDistance  latitudinalMeters,
CLLocationDistance  longitudinalMeters 
)

根据给定的参数构造一个新的 MACoordinateRegion 对象。

参数
centerCoordinate中心点的经纬度坐标。
latitudinalMeters纬度距离，单位为米。
longitudinalMeters经度范围，单位为米。
返回
创建成功则返回 MACoordinateRegion 对象，否则返回 nil。
注解
以下代码示范如何使用该方法。
//创建一个以center为中心，上下各1000米，左右各1000米得区域
MACoordinateRegion viewRegion = MACoordinateRegionMakeWithDistance(center,2000, 2000);
//viewRegion是一个矩形，却不符合 MapView 的横纵比例，需进行调整
MACoordinateRegion adjustedRegion = [mapView regionThatFits:viewRegion];
//显示该区域
[mapView setRegion:adjustedRegion animated:YES];
