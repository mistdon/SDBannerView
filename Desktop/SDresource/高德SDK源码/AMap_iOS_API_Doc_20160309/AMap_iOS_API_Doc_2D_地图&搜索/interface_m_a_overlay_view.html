参数
textureIDOpenGLES 纹理ID。
pointsOpenGLES坐标系点指针，纹理矩形的四个顶点坐标,其第一个坐标为图片左上角，依次顺时针传入其他顶点，参考 glPointForMapPoint: 和 glPointsForMapPoints:count: 方法。
