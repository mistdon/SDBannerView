详细描述

地图视图类（用于地图窗口显示和对地图进行相关的操作）。

注解
以下代码示范如何创建 MAMapView 对象。
MAMapView *mapview = [[MAMapView alloc] initWithFrame:self.view.bounds];
//地图类型为标准地图。如果设置为 MAMapTypeSatellite，则显示卫星地图。
mapview.mapType = MAMapTypeStandard;
//实时交通，默认不显示。如果设置为YES，表示显示实时交通。
mapview.showTraffic = NO;
