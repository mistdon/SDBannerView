文件列表
这里列出了所有文档化的文件，并附带简要说明:
AMapCommonObj.h用于定义了搜索结果的基础数据类型。
AMapNearbySearchManager.h用于完成附近的上传功能。
AMapNearbyUploadInfo.h用于定义附近功能上传对象。
AMapSearchAPI.h用于定义查询相关接口以及回调函数。
AMapSearchError.h用于定义搜索发生的错误。
AMapSearchKit.h用于统一 import 其它搜索相关 .h 头文件。
AMapSearchObj.h用于定义了搜索请求参数和返回对象。
AMapSearchServices.h用于配置搜索所需key，以及获取搜索SDK的版本号。
MAAnnotation.h标注的协议，用于给地图视图提供标注的基本信息。
MAAnnotationView.h用于定义标注视图对象 MAAnnotationView 。
MACircle.h用于定义圆对象 MACircle。
MACircleRenderer.h该类是MACircle的显示圆Renderer。
MACircleView.h用于定义圆视图对象 MACircleView 。
MAGeodesicPolyline.h用于定义 MAGeodesicPolyline 类。
MAGeometry.h用于定义数据类型。
MAGroundOverlay.h用于定义 MAGroundOverlay 类。
MAGroundOverlayRenderer.h此类是将MAGroundOverlay中的覆盖图片显示在地图上的renderer。
MAGroundOverlayView.h用于定义覆盖图片视图对象 MAGroundOverlayView 。
MAHeatMapTileOverlay.h提供热力图图层。
MALineDrawType.h用于定义线 Join 和 Cap 类型。
MAMapKit.h用于统一 import 其它 .h 头文件。
MAMapServices.h用于定义 MAMapServices 类。
MAMapStatus.h记录地图的各种状态，包括：地图中心点、缩放级别、旋转角度等等。
MAMapURLSearch.h用于调起高德地图 App 进行搜索。
MAMapURLSearchConfig.h提供调起高德地图App进行搜索（包括：驾车导航、路径规划、POI搜索）的配置信息。
MAMapURLSearchType.h定义了调起高德地图App进行路径规划的一些常量（规划类型、驾车或公交策略）。
MAMapView.h用于定义地图视图类。
MAMultiColoredPolylineRenderer.h绘制MAMultiPolyline对应的多段线，支持分段颜色绘制。
MAMultiPoint.h用于定义 MAMultiPoint 类。
MAMultiPolyline.h用于定义一个由多个点相连的多段线，绘制时支持分段采用不同颜色（纹理）绘制，点与点之间尾部相连但第一点与最后一个点不相连, 通常MAMultiPolyline是MAMultiColoredPolylineRenderer（分段颜色绘制）或MAMultiTexturePolylineRenderer（分段纹理绘制）的model。
MAMultiTexturePolylineRenderer.h用于绘制MAMultiPolyline对应的多段线，支持分段纹理绘制。
MAOfflineCity.h用于定义离线数据城市信息。
MAOfflineItem.h定义了离线地图下载项的基本信息。
MAOfflineItemCommonCity.h定义了离线地图数据的普通城市对象 MAOfflineItemCommonCity 。
MAOfflineItemMunicipality.h定义了离线地图数据的直辖市对象 MAOfflineItemMunicipality 。
MAOfflineItemNationWide.h定义了离线地图数据的全国概要图对象 MAOfflineItemNationWide 。
MAOfflineMap.h用于定义离线地图下载功能。
MAOfflineProvince.h用于定义离线数据省信息。
MAOverlay.h用于定义地图覆盖物。
MAOverlayPathRenderer.h该类设置overlay绘制的属性，可以使用该类的子类MACircleRenderer, MAPolylineRenderer, MAPolygonRenderer或者继承该类。
MAOverlayPathView.h用于定义 MAOverlayPathView 类。
MAOverlayRenderer.h该类是地图覆盖物Renderer的基类, 提供绘制overlay的接口但并无实际的实现。
MAOverlayView.h用于定义地图覆盖物视图对象 MAOverlayView 。
MAPinAnnotationView.h用于定义类似大头针效果的标注视图对象 MAPinAnnotationView 。
MAPointAnnotation.h标注类，该类定义了一个位于指定位置的标注对象。当需要在地图上添加一个带有标题的点时，可以使用该类而不需要自己定义。
MAPolygon.h用于定义多边形对象 MAPolygon 。
MAPolygonRenderer.h此类用于绘制MAPolygon,可以通过MAOverlayPathRenderer修改其fill和stroke attributes 。
MAPolygonView.h用于定义多边形视图对象 MAPolygonView 。
MAPolyline.h用于定义多段线对象 MAPolyline 。
MAPolylineRenderer.h用于绘制MAPolyline,可以通过MAOverlayPathRenderer修改其fill和stroke attributes 。
MAPolylineView.h用于定义多段线视图对象 MAPolylineView 。
MAShape.h用于定义标注的基本属性和行为。
MATileOverlay.h用于定义添加到基础底层地图的图片集合 MATileOverlay。
MATileOverlayRenderer.h此类是将MAOverlayRenderer中的覆盖tiles显示在地图上的Renderer。
MATileOverlayView.h用于定义基础底层地图图片集合的视图对象 MATileOverlayView 。
MATouchPoi.h用于定义地图上某个图标对应的简要信息。
MAUserLocation.h用于定义定位信息。
