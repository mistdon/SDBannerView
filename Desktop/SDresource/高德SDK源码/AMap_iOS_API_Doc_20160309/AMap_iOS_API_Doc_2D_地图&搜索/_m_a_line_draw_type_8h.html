枚举类型说明

enum MALineCapType

定义了线段端点类型。

枚举值
kMALineCapButt 

端点直接切断。

kMALineCapSquare 

半线宽单位延伸的方块端点。

kMALineCapArrow 

箭头端点。

kMALineCapRound 

以线宽为直接的圆形端点。

enum MALineJoinType

定义了线段连接类型。

枚举值
kMALineJoinBevel 

斜角，用直线连接两条线的外角。

kMALineJoinMiter 

拼接，延伸外角直至两线相接。

kMALineJoinRound 

圆形，半线宽单位的圆形封顶。
