- (id) initWithURLTemplate: (NSString *)  URLTemplate

根据指定的URLTemplate生成tileOverlay

参数
URLTemplate是一个包含"{x}"，"{y}"，"{z}"，"{scale}"的字符串，"{x}"，"{y}"，"{z}"，"{scale}"会被 tile path 的值所替换，并生成用来加载tile图片数据的 URL。例如： http://server/path?x={x}&y={y}&z={z}&scale={scale}。
返回
以指定的URLTemplate字符串生成 MATileOverlay 。
