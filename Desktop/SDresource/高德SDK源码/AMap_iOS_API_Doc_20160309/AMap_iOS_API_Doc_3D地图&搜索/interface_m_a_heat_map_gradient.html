属性

NSArray * colors
 渐变色用到的所有颜色数组，数组中存放 UIColor 对象，默认为 [blue,green,red]。
 
NSArray * startPoints
 每一个颜色对应的起始点数组，默认为 [@(0.2),@(0.5),@(0,9)]。startPoints 与 colors 一一对应，例如：colors 为 [green,red]，则 startPoints 为 [@(0.2),@(1.0)]。
